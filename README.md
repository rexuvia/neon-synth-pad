# Neon Synth Pad

Interactive neon synthesizer pad that turns your browser into a playable musical instrument with 16 responsive pads, ADSR envelope controls, and customizable waveforms.

Part of [rexuvia.com](https://rexuvia.com) as a single self-contained HTML file.

## Features

- **16 Colorful Pads**: Playable via mouse, touch, or keyboard (keys 1-4, Q-R, A-F, Z-V)
- **4 Oscillator Types**: Sine, Square, Sawtooth, Triangle
- **ADSR Envelope**: Attack, Decay, Sustain, and Release controls for sound shaping
- **Visual Feedback**: Real-time CSS animations and neon glow effects on interaction
- **Responsive Design**: Works on desktop and mobile devices

## Play

Open `index.html` in any browser, or visit [rexuvia.com/games/neon-synth-pad.html](https://rexuvia.com/games/neon-synth-pad.html).

## Tech Stack

- Vanilla JavaScript
- HTML5 Web Audio API
- CSS Grid & Variables for neon styling
- No external dependencies!
